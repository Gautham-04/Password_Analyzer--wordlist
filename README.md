Password Strength Analyzer with Custom Wordlist Generator
=========================================================
Features:
- Analyze user passwords using zxcvbn for strength, time-to-crack, and feedback.
- Collect user details (name, birth year, pet name, favorite word, etc.).
- Generate a custom wordlist using those details, leetspeak substitutions, and appended numbers/years.
- Save generated wordlist to custom_wordlist.txt.
- GUI interface using Tkinter for details input, analysis, and wordlist generation.
- (Optional) CLI support via argparse.
